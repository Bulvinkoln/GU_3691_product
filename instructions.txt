# Комнады Git

**git log --oneline** -лог без лишней информации

**git commit -am** “сообщение” -коммит с комментарием плюс add (работает только с теми файлами которые были заэдины ранее)

**git add .** с точкой-добавить все файлы в данной папке

**git add** <имя_файла> -добавить в конкретный файл

**git commit -m** "<комментарий>” -коммит с комментарием 

**git init** -инициализировать папку (инициализация локального репозитория)

**git checkout** - (4 символа коммита) -вернуться к нужному коммиту (или нужной ветке)

**cd (имя нужного файла)**  - переход к нужному файлу (папке)

**git commit --amend -m"новый комментарий”** -меняем комментарий в последнем комимте 

**git log**-все изменения

**git diff** -разница между текущим файлом и закомиченым

**git checkout -b “название ветки**” создание и переход на новую веткупш
